# Scripts

## Topic Create

```bash
chmod +x kafka_topics_create.sh
```

## Kafka Topics

```bash
# Create topics
npm run kafka:create-topics

# List topics
npm run kafka:list-topics

# Delete topics if needed
npm run kafka:delete-topics
```
